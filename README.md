## 🔍 Overview

<a href="#boltAI">BoltAI</a> serves Mac users who want AI assistance integrated into their native Mac workflow rather than browser-based — developers who want AI code assistance without leaving their editor context, writers who want AI writing support in their document workflow, and power users who want fast AI access from anywhere on the Mac.

The <a href="#boltAI">developer AI assistance use case</a> uses BoltAI's quick-access panel to query AI about code from any editor — asking about an error message, requesting an explanation of a code pattern, getting a function implementation suggestion — without switching to a browser and losing the editor context. Custom code assistant personas with project-specific context provide more relevant responses than generic AI chat. The <a href="#boltAI">writer and content creator use case</a> applies AI assistance to writing work in any Mac application — selected text in a document passes to BoltAI for improvement suggestions, style adjustments, or expansion without leaving the document application.

The <a href="#boltAI">research and knowledge work use case</a> uses different AI providers for different query types — a model with web search for current information, a reasoning model for complex analysis, a fast model for quick lookups — all accessible through BoltAI's unified interface without managing multiple browser sessions. <a href="#boltAI">Privacy use case with local models</a> runs AI assistance through Ollama's local models for work that shouldn't leave the Mac.

<a href="#boltAI">Prompt library and templates</a> save frequently used prompts for reuse — building a personal prompt library of effective AI instructions that produce consistent results for recurring task types without rewriting prompts each time. <a href="#boltAI">Markdown rendering in responses</a> displays AI responses with proper formatting — code blocks, headers, bullet lists, and other markdown rendered correctly rather than as raw markup.

<a href="#boltAI">Image input support</a> passes screenshots and images to vision-capable AI models — visual content analysis, screenshot explanation, and image-based queries through BoltAI's interface. <a href="#boltAI">Active development</a> adds new AI provider integrations and Mac-specific features as the AI landscape evolves — BoltAI's provider support grows as new models and services become relevant.

## 🤔 FAQ

<details>
<summary>Can BoltAI switch between AI models per conversation?</summary>

Yes. Model selection is available per-conversation so different models suit different task requirements.

</details>

<details>
<summary>Does BoltAI have a prompt library?</summary>

Yes. Frequently used prompts save for reuse — a personal library of effective AI instructions for recurring tasks.

</details>

<details>
<summary>How does BoltAI handle API keys?</summary>

BoltAI connects to AI providers using the user's own API keys for each provider. Check BoltAI documentation for setup instructions for each provider.

</details>

<details>
<summary>Does BoltAI work with Ollama?</summary>

Yes. Ollama integration connects to locally running models. Ollama must be installed separately and running for local model access.

</details>

<details>
<summary>Does BoltAI run on Apple Silicon?</summary>

Yes. Current versions support Apple Silicon Macs natively.

</details>
